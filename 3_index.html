ホームページ名の変更

2021.3.31Websiteだらだら
github pagesに自分のホームページを公開してから2週間ほどが経過した. 公開当初はホームページ名をkaz.comとしていこうと考えていたが， Google analyticsなどの登録をしているうちにドメインである.comを名前に使用することのバツの悪さに気づいてしまった。 というのも，Googleの検索ページに「kaz.com」と検索をかけると， ほとんどの確率でどこの国のものかわからないホームページに誘導されてしまうのだ。

www.kaz.comのサイト. Helen of Troyという企業らしい

Googleに登録できたときの喜びもあり，はじめに検索をかけたときの驚きはかなりのものであったが，よくよく考えてみると，自分のホームページ名に ドメイン名を含めること自体がいけなかったのだと理解できた。必ず飛ばされる例のサイトにはどうやらkaz.comというドメインが使われていたようなのだ。 そこで，ホームページの名前を変えようと思ったのだが，何せ，このホームページをつくる際，かなり気に入って「kaz.com」と名付けたこともあり， なかなか，次の名前を決めるのには時間を要した。新しい名前の候補としては自身のインスタのアカ名から取って「sawakaz」としたり，"sawa"と"Life huck"の"huck"を取って「sawahuck」 などがあったが，いずれもなんか変な感じがしたし，実際に変更して見てみると，デザイン性にかけているような感じがしたので，却下となった。 それで，また，一日か二日間ほど考え直していたのだが，結局ホームページを公開している時点で誰かにみられる可能性があるということだから， それだったら覚えてもらえやすそうなものにした方がいいだろうということとなり，だったら，名前の意味よりかはテンポの良さを重視する方が覚えてもらうときに覚えてもらいやすくなるなと考えるようになった。 まあそれで，その後もいろいろとこう，グダグダとそしてだらだらと考えていき，最終的にSawalife!という名前にすることとなった。 最後の!だが，これは完全に気分で付けただけである。強調したいとかの類ではない。
さて，結構雑な感じここまで来たが，そもそも名前を変更した一番の目的はあくまでもGoogle検索でいきなりわけのわからない別サイトに飛ばされないようにすることである。 果たして名前を変更してこれが解決したのかは分からないが，また，後日調べてみようと思う。
